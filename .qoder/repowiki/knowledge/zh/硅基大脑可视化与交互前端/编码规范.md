- 所有页面与组件均采用内联 CSS 样式对象（`React.CSSProperties`）进行 UI 定义，未使用 CSS Modules、Sass 或 Tailwind 等外部样式方案。
- API 请求统一通过 `src/api.ts` 中的 `request` 辅助函数发起，该函数自动从 LocalStorage 读取 Token 并注入 `Authorization` 头，同时处理 401 未授权状态下的本地数据清理。
- 复杂页面（如 `BrainView` 和 `ObserverPanel`）内部通过 `setInterval` 实现数据轮询（Polling），并在 `useEffect` 的清理函数中通过标志位（如 `aliveRef`）防止组件卸载后的状态更新。
- 类型定义集中在 `src/types.ts`，并在 API 响应处理中进行显式类型断言或结构化转换（如 `getKnowledgeGraph` 中对后端字段名的规范化映射）。
