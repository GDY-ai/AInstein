- **技术栈**: 采用 Vite + React 18 + TypeScript 构建，使用 React Router DOM 进行客户端路由管理，D3.js 实现复杂的力导向图可视化。
- **分层结构**: `src/api.ts` 封装统一的 RESTful 请求层（含 Token 自动注入与错误处理）；`src/types.ts` 定义全模块共享的 TypeScript 接口；`src/pages/` 承载业务页面逻辑（如 `BrainView` 负责图谱渲染，`Login` 负责认证）；`src/components/` 存放可复用 UI 组件（如 `ObserverPanel`）。
- **核心交互**: `BrainView.tsx` 通过 D3 Simulation 实现节点动态布局与拖拽交互，并集成 10 秒轮询机制实现图谱数据的准实时更新；`ObserverPanel.tsx` 以折叠面板形式展示 AI 生成的观察员日志，支持历史回溯与手动触发总结生成。
- **状态管理**: 采用 React Hooks (`useState`, `useEffect`) 结合 LocalStorage 进行轻量级状态持久化（用户信息与 Token），未引入 Redux 等重型状态库。
