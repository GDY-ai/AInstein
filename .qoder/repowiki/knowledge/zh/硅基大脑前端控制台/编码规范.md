- 所有 API 请求统一通过 `src/api.ts` 中的 `request` 辅助函数发起，自动注入 Bearer Token 并处理 401 未授权状态。
- 页面组件普遍采用内联 CSS 样式对象（`CSSProperties`）进行视觉设计，而非引入外部 CSS 文件或 CSS-in-JS 库。
- 受保护的路由页面在组件入口处校验 `getToken()`，若缺失则通过 `useNavigate` 强制重定向至登录页。
- 复杂页面（如 `BrainList`）将 UI 拆分为多个小型函数式子组件（如 `BrainCard`, `AdminHero`），并在同一文件内定义以保持上下文紧凑。
