- **技术栈与构建**: 采用 `Vite` + `React` + `TypeScript` 架构，使用 `react-router-dom` 进行客户端路由管理，`d3` 用于数据可视化（如认知图谱）。
- **分层结构**: 
  - `src/pages/`: 页面级组件，负责路由视图渲染（如 `BrainList`, `BrainView`, `Login`）。
  - `src/components/`: 可复用 UI 组件（如 `ObserverPanel` 观察员面板）。
  - `src/api.ts`: 统一的 API 通信层，封装 `fetch` 请求、Token 管理及错误处理，作为前端与后端的唯一边界。
  - `src/types.ts`: 全局 TypeScript 类型定义，确保前后端数据结构一致性。
- **路由与权限**: `App.tsx` 定义路由表，通过 `RequireAuth` 高阶组件实现基于 Token 的路由守卫，区分公开页面（Login）与受保护页面（BrainList, BrainView）。
- **状态管理**: 采用轻量级策略，通过 `localStorage` 持久化用户会话（Token/User），组件内部使用 `useState/useEffect` 管理本地状态，无引入 Redux/Zustand 等全局状态库。
