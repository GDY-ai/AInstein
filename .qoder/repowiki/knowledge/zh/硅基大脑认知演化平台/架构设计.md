- **统一 API 网关契约**：`frontend/src/api.ts` 作为唯一后端通信入口，封装了 Auth、Brain 生命周期、知识图谱及观察员日志等所有领域接口，并统一处理 JWT Token 注入与 401 失效清理。
- **多态视图路由编排**：`App.tsx` 基于 `RequireAuth` 守卫进行路由分发，根据用户角色（Admin/User）在 `BrainList` 中动态渲染“上帝视角”或“个人工作台”，并隔离出 `BigScreen` 沉浸式大屏与 `BrainView` 深度分析页。
- **跨模块状态同步机制**：各视图组件（如 `BrainView`、`ObserverPanel`）独立维护轮询逻辑（Polling），通过共享的 `api` 客户端获取实时认知状态（Thinking/Paused/Completed），实现图谱演化与观察员叙事的异步解耦更新。
- **类型系统全域共享**：`frontend/src/types.ts` 定义了从底层认知元素（CognitiveNode）到高层业务实体（Brain, User）的统一数据结构，确保图谱渲染、列表展示与大屏拓扑之间的数据语义一致性。
