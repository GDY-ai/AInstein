- 所有 API 请求通过 `api.ts` 中的 `request` 辅助函数统一发起，自动携带 Bearer Token 并处理 JSON 序列化。
- 采用 CSS-in-JS (Inline Styles) 模式，所有组件样式直接定义在 `.tsx` 文件末尾或内部，不依赖外部 CSS 文件。
- 认知节点类型（CE Type）与状态（State）使用统一的常量映射表（如 `CE_COLORS`, `STATE_LABEL`）进行 UI 渲染，确保跨页面视觉一致性。
- 复杂可视化组件（如 BrainView, BigScreen）采用 D3/Canvas 命令式绘图与 React 声明式状态分离的模式，通过 `useRef` 托管图形实例。
