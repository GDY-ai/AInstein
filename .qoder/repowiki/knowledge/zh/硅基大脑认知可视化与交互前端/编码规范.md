- API 请求统一通过 `src/api.ts` 中的 `request` 辅助函数发起，自动处理 `Authorization` Bearer Token 注入及 401 状态下的本地存储清理。
- 页面级组件（如 `BrainView`, `ObserverPanel`）普遍采用 `useState` 管理数据状态，并结合 `useEffect` 实现定时轮询（Polling）以同步后端最新认知状态。
- 复杂可视化逻辑（如 D3 力导向图）直接在 React 组件内通过 `useRef` 操作 DOM 元素，避免 React 重渲染干扰 D3 的物理仿真过程。
- 类型定义集中在 `src/types.ts`，涵盖从基础用户信息到复杂的认知节点（CognitiveNode）及观察员日志（ObserverLog）结构，确保全链路类型安全。
