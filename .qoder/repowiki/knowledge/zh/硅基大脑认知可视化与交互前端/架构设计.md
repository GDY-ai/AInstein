- **技术栈**: 采用 Vite + React 18 + TypeScript 构建，使用 React Router DOM 进行客户端路由管理，D3.js 实现复杂的力导向图可视化。
- **分层结构**: `src/api.ts` 封装统一的 RESTful API 请求层（含 Token 自动注入与错误处理）；`src/types.ts` 定义全局 TypeScript 接口；`src/pages/` 承载业务页面（如 `BrainView` 负责图谱渲染，`Login` 负责认证）；`src/components/` 复用 UI 组件（如 `ObserverPanel`）。
- **核心逻辑**: `BrainView.tsx` 内部集成 D3 Simulation 引擎，通过 `useRef` 维持仿真状态，利用 `requestAnimationFrame` 或定时器实现每 10 秒的图谱数据轮询与增量渲染。
- **入口配置**: `main.tsx` 挂载应用并配置 `BrowserRouter` 的 `basename` 为 `/ainstein`，`vite.config.ts` 设定构建输出目录与基础路径。
