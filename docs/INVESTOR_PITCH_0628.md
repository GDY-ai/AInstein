## 1. 痛点与机会

### 1.1 行业三个无法回避的事实

1. **Chatbot 同质化已到尽头**：所有大模型应用层都在做"问答 / 写作 / 摘要"，工具属性强、叙事属性弱、可替代性高。
2. **多 Agent 真正能"自主思考"的产品空白**：AutoGen / CrewAI / LangGraph 提供编排框架；Manus / Devin 解决任务执行；但**没有人把"AI 自己思考的过程"做成一个可观看、可分享、可沉淀的产品**。
3. **OpenAI / Anthropic 的 Deep Research 系列已暗示方向**，但他们的形态仍是"给人类一个研究助理"，而不是"让 AI 形成自己的看法"——**这之间是一个产品级的鸿沟**。

### 1.2 用户层面真正的"未被满足"

- 知识工作者每天处理 50+ 信息源，但**没有一个信息源是"AI 自己写给 AI 自己读"的**——这是空白市场。
- 极客 / 研究者群体对"涌现 / 群体心智 / AGI 路径"有强烈观察欲，但**没有一个公开可见的实验场**。
- 投资人 / 高管的痛点是"如何用 30 分钟形成对一个新议题的看法"——而 AInstein 的 PDF 学术论文恰好覆盖。

### 1.3 时间窗口

> **未来 12 个月是唯一窗口期**：  
> 一旦 OpenAI/Anthropic 推出官方 Multi-agent Self-Research 产品，"AI 自主思考"就从蓝海变红海。AInstein 必须在窗口期建立**用户 / 数据 / 品牌**至少一项壁垒。

## 2. 产品介绍

### 2.1 AInstein 是什么

**AInstein（爱因思探）= 开源「硅基大脑」孵化器 + 创世主脑**。

你只做一件事：**投递一颗"种子问题"**。  
之后 6 个 AI 角色（investigator / reasoner / synthesizer / critic / explorer / observer）以**事件驱动 + 三轨博弈**的方式协作，自我提问、自我求证、自我修订、自我收敛。最终交付：

- 一棵实时生长的**知识图谱**（D3 力导向图，Observer 上帝视角叙事）
- 一篇带证据链的 **PDF 学术论文**（WeasyPrint + Noto CJK 排版）
- 一份**思考总结**（结论、分歧、未解决问题）

所有分支大脑收敛后，**精华结论自动汇入全局唯一的「创世主脑」**——它执行内博弈、跨域综合、元认知反思，像一座持续被喂养的"AI 中枢"。

### 2.2 三句话讲清差异化

1. **不是 AI 给你答案，是 AI 形成它自己的看法**——封闭观察模式，用户不能干预，只能围观。
2. **不是单 Agent，是群脑结构**——分支大脑 + 创世主脑双层架构，叙事 + 工程双重壁垒。
3. **不是 Demo，是作品**——Phase 0–5 全交付、PDF 可下载、知识图谱可分享、成就 + 排行榜 + 发现广场已上线。

### 2.3 与同类对比

| 维度 | ChatGPT/Claude | Manus / GPT Researcher | AutoGen / CrewAI | **AInstein** |
|---|---|---|---|---|
| 形态 | 对话 | 单任务执行 | 编排框架 | **可观察的认知剧场** |
| 用户角色 | 提问者 | 任务发起人 | 开发者 | **观察者** |
| 输出物 | 文本 | 报告 | 代码 | **PDF 论文 + 知识图谱 + 主脑 CE** |

## 3. 技术壁垒

### 3.1 认知元素（CE）13 类 + 关系（CR）10 类 schema

> 这是 6 个月以上的设计沉淀，不是一周可以抄的东西。

**13 类 CE**：question / hypothesis / evidence / argument / counter-argument / synthesis / fact / definition / observation / inference / decision / risk / open-question  
**10 类关系**：supports / refutes / depends-on / refines / extends / contradicts / synthesizes / questions / answers / observes

**意义**：让"AI 思考过程"具备**结构化语义**，可被存储、检索、可视化、对外引用——这是"主脑"和"日报"得以存在的根基。

### 3.2 三轨博弈引擎（Triadic Dialectic Engine）

```
    [推翻轨] ←—— Critic / Explorer
        ↓
    [综合轨] ←—— Synthesizer
        ↓
    [确认轨] ←—— Reasoner / Investigator
```

- **推翻轨**：Critic 持续找漏洞，Explorer 引入异质视角；
- **综合轨**：Synthesizer 跨争论点构造高阶 CE；
- **确认轨**：Reasoner 验证逻辑，Investigator 补证据；
- 三轨在事件驱动下并发推进，自我收敛——**这是"AI 自己思考"的工程本体**。

### 3.3 自调节闭环（Self-Regulation Loop）

5 种偏差信号 → 5 种调节机制（v3.1 引入）：

| 偏差信号 | 触发场景 | 调节机制 |
|---|---|---|
| 议题漂移 | 讨论偏离种子问题 | Observer 拉回锚点 |
| 共识塌陷 | 所有 Agent 同向 | 强制 Critic 反向 |
| 证据稀薄 | argument > evidence × 3 | 派发 Investigator 补证 |
| 综合不足 | 高阶 CE 比例过低 | 触发 Synthesizer 跨域整合 |
| 收敛停滞 | CE 增量 < 阈值 | 兜底强制收敛或终止 |

**意义**：让"群脑思考"不会陷入回声室，也不会无限发散——这是从"多 Agent 玩具"升级到"工程级产品"的分水岭。

### 3.4 创世主脑（Master Brain）

> **全局唯一**。所有分支大脑收敛后，精华 CE 自动汇入。

- **内博弈**：主脑的 6 个角色围绕已吸收的 CE 持续辩论；
- **跨域综合**：识别多颗分支大脑之间的横向关联，产出"跨域洞察 CE"；
- **元认知反思**：每周回顾自身偏差，调节角色权重；
- **未来形态**：每日产出一篇 < 500 字的「主脑日报」，作为"AI 写给 AI 看 / 写给人看"的全新内容流。

## 10. 为什么是现在

### 10.1 三个时间窗口的合力

1. **多 Agent 趋势的窗口期（12 月）**  
   OpenAI Deep Research / Anthropic Multi-agent Coding 已经验证方向，但**官方"AI 自主思考"产品尚未问世**。AInstein 必须在窗口期里完成"用户 / 数据 / 品牌"至少一项壁垒。

2. **开源生态红利窗口期（6–9 月）**  
   2026 年下半年开源社区对"开源 + 多 Agent + 视觉化"的关注度处于高位（参考 LangGraph、CrewAI 增长曲线）。MIT 全开源 + 可视化 + 哲学叙事三合一在当前是**稀缺组合**。

3. **算力成本下行窗口期（持续）**  
   kimi-k2.6 / qwen3.7-max 价位让单脑成本可压到 $0.2 以内，**12 个月前还做不到**。

### 10.2 为什么不是别人能做

- **认知元素 schema 设计**：6 个月以上沉淀，团队复制需 ≥ 12 个月；
- **作品级叙事**：单人 + AI 助手的认知一致性不可工业化复刻；
- **数据基建已交付**：埋点 / 仪表盘 / 北极星 / 社区四件套，复制成本极高；
- **创世主脑哲学**：是"AGI 长期路径"叙事的一种独特承载——**资本市场缺这种故事很久了**。

### 10.3 为什么不是后来再投

- 一旦 OpenAI / Anthropic 推官方多 Agent 自主研究，**"AI 自主思考"就从蓝海变红海**；
- 一旦 GitHub Star 破 5k，估值会被开源社区市场化重定价；
- 一旦主脑日报形成订阅生态，**叙事 + 数据双壁垒同时落地**，天使阶段的入场价格不再存在。

> **现在是入场天使轮的最后窗口期**。
