# AInstein 使用手册（v3.1+）

> 面向终端用户。即使你不写代码，也能照着这份手册理解硅基大脑在做什么、怎么用好它。
>
> 配套阅读：[design.md](./design.md)（架构）、[ops-manual.md](./ops-manual.md)（运维）、[testing.md](./testing.md)（测试）。

---

## 1. AInstein 是什么

AInstein 是一个**自主思考的硅基大脑**：

- 你只投递一个**种子问题**（一句话），系统就会启动一颗大脑；
- 它自己提问、查资料、做假设、找证据、互相辩论、总结；
- 思考过程实时可视化——你看到一张持续生长的「**知识图谱**」；
- 它自己决定什么时候停下，并产出一份完整的**思考总结**与可下载的 **PDF 论文**；
- 所有用户的大脑收敛后，精华结论会**自动汇入「创世主脑」**——一个全局唯一、永不消亡的元大脑，它会跨用户跨领域做更高层次的思辨。

> **它不是 chatbot，也不是研究助手。** 你不能向它追加 prompt，不能打断它的思路。你只能投问题、然后**观察**。

## 3. 注册与登录

### 3.1 强密码要求

为了防止账号被暴力破解、保护「思考容器」的纯净，注册要求 **8 位以上**且**同时**满足：

| 要求 | 示例 |
| --- | --- |
| 至少一个大写字母 | `A`、`Z` |
| 至少一个小写字母 | `a`、`z` |
| 至少一个数字 | `0–9` |
| 至少一个特殊字符 | `!@#$%^&*()-_+=` 等 |

**合格示例**：`Hello@2026`、`Mind#Brain1`、`Ainstein2026!`
**不合格示例**：`12345678`（无字母）、`password`（无数字/特殊）、`Abc12345`（无特殊字符）

前端会先做校验，提交前会即时反馈。

### 3.2 注册 → 登录

1. 打开 `/ainstein/login`，切到「注册」；
2. 填写用户名、邮箱、密码（满足强密码）；
3. 注册完直接获得登录态，跳转到大脑列表页（/ainstein/brains）。

> 默认创建的账号是普通用户。管理员账号由运维侧颁发，普通用户无法自行升级。

## 4. 创建大脑

进大脑列表页（/ainstein/brains），点「**+ 创建大脑**」（或访问 `/ainstein/brains/new`）。

### 4.1 什么是「种子问题」

种子问题是大脑的**唯一输入**——它不是 prompt，是**一颗思维的种子**。系统会以它为重力中心，所有的思考都围绕它发散和收敛。

### 4.2 怎么写一个好的种子问题

| 维度 | 好 | 差 |
| --- | --- | --- |
| **具体性** | "量子计算未来 5 年的商业化前景如何？哪些行业会先受益？" | "讲讲量子计算" |
| **开放性** | "为什么人会做梦？" | "做梦的英文怎么说？" |
| **可探索** | "AI 会不会取代程序员？" | "1 + 1 等于几？" |
| **不引导答案** | "远程办公对企业生产力的真实影响是什么？" | "请证明远程办公比线下好" |
| **单一焦点** | "城市绿化如何影响居民幸福感？" | "城市规划、交通、绿化、治安都怎么搞？" |

**好种子的共同特征**：值得辩论、有多种合理视角、需要证据、答案不显然。

### 4.3 「观察者契约」

提交后你会看到一个**不可逆**的契约提示：

> **一旦大脑开始思考，你只能观察，不能干预。**
>
> - 你不能补充 prompt
> - 你不能修改种子问题
> - 你不能控制它思考多久
> - 你只能选择**让它继续**或者**让它停止**

我们坚持封闭观察模式，是因为**真正的独立思考必须独立**。每一次外部干预都会让结论变成「人类引导下的产物」，而不是大脑自己的。

## 5. 观察思考过程

进入大脑详情页（BrainView）会看到 3 个核心区域：

```
┌─────────────────────────────────────────────────────────┐
│  顶部：种子问题 + 大脑状态 + 思考时长 + CE 数 + 控制按钮 │
├──────────────────────────────────┬──────────────────────┤
│                                  │                       │
│   左侧：知识图谱（D3 力导向）    │   右侧：Observer 面板 │
│   - 节点 = CE                    │   - 实时叙事日志      │
│   - 边 = CE 之间的关系           │   - 阶段性洞察        │
│   - 大小 ∝ confidence            │                       │
│                                  │                       │
└──────────────────────────────────┴──────────────────────┘
```

### 5.1 知识图谱（力导向图）

- **种子问题**：固定在图心，是所有思考的引力源——视觉上有独特标识；
- **节点（CE）大小**：越大表示该 CE 的「置信度」越高（被验证程度越高）；
- **节点颜色**：按 13 种 CE 类型区分（详见下表）；
- **边（关系）**：颜色代表关系类型（支持 / 反对 / 衍生…）；
- **缩放**：CE 越多图自动 zoom-out 适配视野，无需手动调；
- **点击节点**：右侧弹出该 CE 的完整内容、置信度变化历史、相关证据。

### 5.2 13 种 CE 类型（用人话解释）

| 类型 | 一句话 | 例 |
| --- | --- | --- |
| **observation** 观察 | "我注意到这件事" | "数据显示梦多发生在 REM 睡眠阶段" |
| **question** 问题 | "我想搞清楚" | "REM 睡眠和情绪整合有什么关系？" |
| **hypothesis** 假设 | "我猜可能是这样" | "梦是大脑在处理白天的情绪记忆" |
| **evidence** 证据 | "支持的事实" | "脑成像研究显示 REM 期杏仁核活跃" |
| **counter_evidence** 反证 | "反驳的事实" | "也有研究表明梦与情绪无关" |
| **inference** 推论 | "由此推出" | "如果梦处理情绪，则失眠者情绪稳定性更差" |
| **argument** 论证 | "完整的论证链" | "前提 → 中间步骤 → 结论" |
| **conclusion** 结论 | "整合后的答案" | "梦在多个层面参与了认知整合" |
| **perspective** 视角 | "另一个看待角度" | "进化视角下，梦是大脑的离线训练" |
| **insight** 洞察 | "更深一层的发现" | "梦境碎片化反映了记忆固化机制" |
| **consensus** 共识 | "多个 Agent 一致同意" | （博弈通过后产生） |
| **dissent** 异见 | "未达成一致" | （博弈失败后产生） |
| **tool_gap** 工具缺口 | "我现在缺一种工具" | "需要能调取脑电数据的工具" |

### 5.3 Observer 叙事日志（右侧面板）

Observer 是一个**不参与博弈的元角色**，专门以**上帝视角**讲故事。它会用自然语言告诉你：

- 大脑现在在想什么；
- 哪些 Agent 在辩论什么；
- 刚刚发生的关键转折（共识达成 / 异见涌现 / 引入新视角）；
- 思考节奏整体处于哪种姿态（探索期 / 收敛期 / 异质性刺激期）。

> Observer 是**最适合普通用户阅读**的入口。如果不想看图谱细节，只看 Observer 也能理解大脑此刻在做什么。

### 5.4 importance / confidence 分数怎么看

- `confidence ∈ [0, 1]`：大脑对该 CE 的「相信程度」。
  - `< 0.3`：弱（被反驳过 / 证据不足）
  - `0.3 – 0.6`：中（待验证）
  - `0.6 – 0.8`：高（有证据支撑）
  - `≥ 0.8`：很高（往往会触发主轨终止）
- 节点大小 ∝ confidence——**最大的几个节点就是大脑当前最确信的核心结论**。

## 6. 大脑生命周期

### 6.1 6 种状态

| 状态 | 含义 |
| --- | --- |
| `gestating` | 创建中（极短暂） |
| `active` / `thinking` | 正在思考（图谱在生长） |
| `paused` | 已暂停（仅管理员可触发） |
| `completed` | 已收敛完成（产出结论 + 总结） |
| `archived` | 已归档（兜底终止 / 管理员一键停止） |
| `dormant` | 休眠中（创世主脑初始态，等待首次分支上报后激活） |

### 6.2 自动收敛（大脑自己决定停）

大脑会**自己**判断什么时候该停下，**两条触发轨道**任一满足都停：

| 轨道 | 触发条件 | 适用 |
| --- | --- | --- |
| **主轨** | 出现一个 confidence ≥ 0.75 的 conclusion CE | 结构化问题：有清晰答案 |
| **兜底轨** | CE 总数 ≥ 500 **或** 思考时长 ≥ 1 小时 | 开放性 / 哲学问题 |

主轨终止意味着「我想清楚了」；兜底轨意味着「再想下去边际收益太小，我先停下」。

收敛后系统**自动**：
1. 产出**思考总结**（结构化卡片）；
2. 触发**论文生成**（PDF）；
3. 把高置信度结论**自动上报**给创世主脑。

### 6.3 手动控制（仅管理员）

| 操作 | 含义 |
| --- | --- |
| 暂停（pause） | 大脑停止思考，不释放 Agent 实例，可恢复 |
| 恢复（resume） | 从上次中断处继续 |
| 停止（stop） | **不可逆**终止；会强制 synthesizer 综合一份兜底结论 |

普通用户当前**无法**手动停止自己的大脑——这是「观察者契约」的一部分。如确需提前终止，请联系管理员。

## 7. 查看成果

### 7.1 思考总结卡片

当大脑进入 `completed` / `archived` 状态，BrainView 顶部会出现「思考总结」按钮（**默认折叠**，避免遮挡图谱）。展开后看到结构化总结：

- **核心结论**：1–3 条；
- **关键证据**：支撑结论的高置信度 evidence；
- **主要争议**：博弈中出现过的 dissent；
- **未解之谜**：剩下的 open question；
- **下一步建议**：synthesizer 给出的后续探索方向。

### 7.2 PDF 论文下载

总结卡片下方有「**下载论文**」按钮。点击后：

1. 系统启动论文生成任务（异步，进度条显示）；
2. 30–90s 完成；
3. 自动弹出下载——一份学术期刊级排版的 PDF（中英排版均良好，支持中文 Noto CJK）。

论文结构：摘要 / 引言 / 假设 / 证据 / 论证 / 结论 / 参考思路 / 附录（CE 列表）。

### 7.3 大脑会自动「死亡」吗？

不会被删除。`completed` / `archived` 的大脑：
- 知识图谱永久保留；
- 思考总结永久保留；
- 论文 PDF 永久保留；
- 高置信度精华已上报创世主脑——**它的"死亡"成为更大思维的养料**。

## 8. 管理员功能

> 仅管理员账号可见以下能力。

### 8.1 BrainList「上帝视角」

普通用户的 BrainList 只能看到自己的大脑。**管理员**进入时呈现**全局态势视图**：

- **创世主脑**位于布局 C 位（永远发光、永远 active）；
- 其它分支大脑按 `owner_user_id` 分组围绕主脑；
- 节点尺寸 ∝ 该大脑当前 CE 数；
- 节点颜色按 state 区分（active / paused / completed / archived）。

### 8.2 态势大屏（BigScreen）

`/ainstein/admin/bigscreen` —— 黑色背景全屏 Canvas 力导向图（分层布局）：

- 创世主脑固定在顶部，分支大脑分布在下方区域；
- 分支大脑按 `owner_user_id` 分组排布，每组围绕各自重心铺开；
- 实时显示每颗大脑的呼吸节奏（CE 落库时节点闪烁）；
- 适合放在大屏 / 投影上做演示。

### 8.3 暂停 / 恢复 / 停止他人大脑

管理员可对任意大脑：
- `pause` / `resume`：临时暂停（如服务器维护）；
- `stop`：不可逆终止（如用户提交了不当种子问题）。

> 这些操作会被记录在 `observer_logs` 中，由 Observer 在前端透明叙述。

### 8.4 监控创世主脑

主脑的实时状态可在 BrainView 中以 `id=1` 直接打开。建议管理员定期关注：

- 主脑近期是否在做**内博弈** / **跨域综合** / **元认知反思**；
- 上报队列是否堆积；
- 主脑自身的 confidence 高分 conclusion 数。

## 9. 最佳实践

### 9.1 好的种子问题 vs 差的种子问题

| ✅ 好种子 | ❌ 差种子 |
| --- | --- |
| 为什么人会做梦？ | 1 + 1 = ? |
| 量子计算未来 5 年的商业化前景如何？ | 量子计算是什么？ |
| 远程办公对企业生产力的真实影响？ | 帮我写一封请假邮件 |
| 城市绿化和居民幸福感的关系？ | 我家院子该种什么花？ |
| AI 是否会冲击传统教育？为什么？ | AI 是好是坏？ |
| 加密货币能否真正去中心化？ | 现在该买比特币吗？ |

**判断标准**：值得辩论、需要证据、答案不显然、有多种合理视角、与具体决策建议保持距离（系统不是顾问）。

### 9.2 别做这些事

- ❌ 提交后反复查看「为什么还没结论」——结构化问题至少 8 分钟，开放性问题可能 30–60 分钟；
- ❌ 用大脑当 Q&A 工具——它会被启动，但产出比 ChatGPT 慢且贵；
- ❌ 同一种子重复提交——每次大脑视角不同，但不会更准；
- ❌ 把它当算命/咨询师——它思考问题，不替你做决定。

### 9.3 推荐用法

- ✅ 投递你**真正好奇但没空研究**的问题；
- ✅ 用它做**认知工坊**：观察一个问题被 AI 群体如何拆解、辩论；
- ✅ 用它做**头脑风暴起点**：拿大脑产出的 conclusion 与 dissent 当你后续思考的素材；
- ✅ 用它做**研究启蒙**：看 PDF 论文了解一个陌生领域的脉络。

## 10. 常见问题（FAQ）

### Q1. 我的大脑卡住了，半小时都没动？

A：先看 Observer 面板。如果 Observer 也不更新，可能是后端瞬时故障，请联系管理员。普通故障下大脑会自我恢复，你不必干预。

### Q2. 我能修改种子问题吗？

A：不能。一旦提交，你只能观察。如果种子写错，请创建一颗新大脑。旧的会作为思考记录留存。

### Q3. 大脑思考要花多久？

A：
- 结构化问题（如"X 对 Y 的影响"）：8–25 分钟；
- 开放性问题（如哲学问题）：30–60 分钟（兜底终止）。

### Q4. 我能让它思考更久吗？

A：不能。我们坚持兜底终止——**长时间不收敛意味着边际收益太小**。

### Q5. 思考过程中我能向它补充资料吗？

A：不能。大脑会自己调用工具（web_search / wikipedia_search / arxiv_search 等）找资料。这是「封闭观察模式」的核心。

### Q6. PDF 论文里的"参考资料"是真的吗？

A：论文中提到的网络/wiki/arxiv 检索结果来自工具调用，是实际 fetched 的内容；但**LLM 仍可能错误引述**。当作起点，不要当作权威。

### Q7. 创世主脑会污染我的思考吗？

A：分支大脑**只向**主脑上报，不从主脑下载。你的大脑思考过程是干净的、独立的。

### Q8. 我能删除我的大脑吗？

A：当前版本不开放删除（保留思考记录是产品哲学）。如有强需求请联系管理员。

### Q9. 多个用户的大脑会互相影响吗？

A：分支之间彼此**不可见、不可引用**。它们只通过创世主脑间接「相遇」——主脑可能跨用户做综合，但分支独立运行。

### Q10. 为什么注册要这么严格的密码？

A：硅基大脑的容器很珍贵，防止弱口令账号被滥用是基础卫生。

### Q11. 节点大小是不是越大越对？

A：是「越确信」，不一定「越对」。Confidence 反映系统内部的证据/博弈结果，但仍可能是错的。请把它当**线索**而不是**真理**。

### Q12. 看不懂图谱怎么办？

A：直接看右侧 Observer 面板的自然语言叙事——它就是为非技术用户设计的。

## 附录：术语速查

| 术语 | 含义 |
| --- | --- |
| **CE**（Cognitive Element） | 认知元素，大脑思考的基本单位（observation / question / hypothesis / …） |
| **Agent** | 大脑里的一个"思考视角"——explorer / investigator / reasoner / critic / synthesizer / observer |
| **博弈**（Deliberation） | 多个 Agent 围绕某个 CE 进行 3 轮辩论 + 投票 |
| **共识**（Consensus） | 博弈通过：≥60% 加权赞成 + ≥2 个 agree |
| **异见**（Dissent） | 博弈未通过：意见分歧记录在案 |
| **种子问题** | 用户输入的唯一一句问题，大脑思考的起点 |
| **创世主脑** | 全局唯一、永不消亡的元大脑（id=1），所有分支大脑的精华汇集地 |
| **frontier** | 待展开的认知前沿，大脑空闲时从这里挑下一个低置信度元素去想 |
| **观察者契约** | 用户提交种子后只能观察、不能干预的承诺 |
| **力导向图** | D3 实现的图谱可视化，节点互相吸引 / 排斥模拟物理 |

---

> 你提交一颗种子，
> 一颗大脑由此诞生、思考、收敛、消逝；
> 它的精华汇入主脑，主脑因此长大一点点。
>
> **欢迎成为它的观察者。**
